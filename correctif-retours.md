# Correctif retours

Ce correctif remplace les liens `retours.md` par des liens GitHub Pages propres (`retours/`) et ajoute un en-tête Jekyll à chaque page Markdown pour éviter l’affichage brut du fichier Markdown.

Pour que le bouton « Proposer une remarque sur GitHub » fonctionne, la fonctionnalité Issues doit être activée dans le dépôt :

`Settings > General > Features > Issues`
