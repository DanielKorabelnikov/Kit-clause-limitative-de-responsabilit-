# Kit clause limitative de responsabilité

Ce dépôt contient un mini-guide pratique consacré aux clauses limitatives de responsabilité dans les contrats IT, SaaS et numériques.

Site publié :  
https://danielkorabelnikov.github.io/Kit-clause-limitative-de-responsabilite/

## Contenu

- Notice explicative
- Checklist de revue
- Clauses types et variantes
- Fiche de négociation du plafond
- Sources juridiques utiles
- Page de retour / remarques

## Usage

Ce kit est un support de travail général. Il doit être adapté au contexte du contrat, au niveau de criticité du service, aux obligations essentielles, au rapport de négociation et aux assurances disponibles.

## Publication GitHub Pages

Le dépôt doit être public et GitHub Pages doit être configuré comme suit :

```text
Settings > Pages > Deploy from a branch > main > /(root)
```

Le nom du dépôt doit rester exactement :

```text
Kit-clause-limitative-de-responsabilite
```
